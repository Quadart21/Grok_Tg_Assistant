Как запустить программу

1
Дважды нажмите start.bat в папке программы.
2
Откроется окно с вкладками — это главная панель управления (GUI).
3
Сверху строка статуса: зелёные галочки = всё готово к работе.
Окно программы и эта инструкция работают физически на вашем ПК. Интернет нужен только для Telegram, Grok и прокси.

Вкладка 2 — Подключение

Заполните один раз и нажмите «Сохранить подключение».

Telegram API

  1. Откройте my.telegram.org/apps
  2. Войдите по номеру телефона
  3. Создайте приложение → скопируйте API ID и API Hash

Ключ Grok

  1. Откройте console.x.ai
  2. Создайте API Key → вставьте в поле «Ключ Grok»
Кнопка «Показать / скрыть ключи» — чтобы увидеть введённый пароль и ключ.

Вкладка 3 — Аккаунты и прокси

  1. Нажмите «Открыть папку» — откроется sessions/
  2. Положите туда файлы .session или папки tdata
  3. Нажмите «Обновить список»
  4. Кликните аккаунт в таблице → справа укажите прокси → «Сохранить прокси»

Массовая вставка прокси

Кнопка «Вставить список прокси» — по одной строке на аккаунт:

192.168.1.1:1080 или 192.168.1.1:1080:логин:пароль

Прокси и стиль запоминаются для каждого аккаунта — при следующем запуске подставятся автоматически.

Вкладка 4 — Стиль общения

Опишите, как Grok должен писать — это «роль» или характер переписки.

  • Стиль по умолчанию — для всех аккаунтов без группы
  • Группы — разный стиль для разных аккаунтов (отметьте галочками)

Пример: «Вы менеджер. Пишете вежливо, коротко, без спама.»

Нажмите «Сохранить стили общения».

Вкладка 1 — Рассылка и диалог

  1. Впишите username получателей — по одному на строку (с @ или без)
  2. Включите «Вести диалог» — Grok будет отвечать на сообщения
  3. Включите «Продолжить прошлые диалоги» — после паузы подхватит старые чаты
  4. Нажмите «Запустить»
КнопкаДействие
▶ ЗапуститьПервое сообщение + онлайн-диалог
↻ Продолжить диалогиТолько старые переписки, без новых username
■ ОстановитьПауза — всё сохраняется, можно продолжить позже

Вкладка 5 — Диалоги

Список всех сохранённых переписок. Статус «на паузе» — нажмите «Продолжить диалоги» на вкладке 1.

Данные хранятся в data/state.json на вашем диске.

Частые вопросы

Строка статуса жёлтая?

Не хватает API, ключа Grok или аккаунтов в папке — пройдите шаги 1–3.

Ошибка при установке?

Перенесите папку программы в путь без русских букв, например C:\tg-outreach\

Как открыть эту инструкцию снова?

В окне программы нажмите «📖 Инструкция» вверху — откроется эта же страница в браузере.
